# Personal Website - Frontend

This is the frontend application for the personal website, built with React, Vite, and Three.js.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm start
```

## Deploy to Render

1. Create a new **Static Site** on Render
2. Connect your GitHub repository
3. Set the **Root Directory** to: `frontend`
4. Set the **Build Command** to: `npm install && npm run build`
5. Set the **Publish Directory** to: `dist`

The site will be deployed as a static site with optimal performance.
