# Projects

## VoiceAI Web Agent
A Manifest V3 Chrome extension built with JavaScript and Flask that enables voice-based web interaction. The system captures real-time audio and processes it through asynchronous speech-to-text and text-to-speech pipelines, allowing users to navigate and interact with websites using natural speech.

## Anr Awaaz
An AI-driven pronunciation analysis tool designed for non-literate learners. Built using Whisper and FastAPI, the platform performs phoneme-level analysis and provides targeted feedback with multilingual voice guidance to improve spoken language skills.

## TrackerFlow AI
A workflow automation system consisting of a Chrome extension and a Flask/React backend. The platform converts natural language prompts into structured trackers and dashboards, enabling users to automate repetitive tracking tasks.

## TestGuard Platform
A secure automated testing environment that uses Selenium, WebDriverIO, and Docker sandboxing to validate unit, integration, and regression tests. The system isolates test execution to prevent environment conflicts and improve reliability.

## CrickAI Vision
A probabilistic prediction system that models cricket player shot distributions and delivery types using data analytics and generative AI models. The platform provides insights into player behavior and match strategy.

## Games.Random
A full-stack game development platform built with React and Node.js that allows real-time game design, modification, and AI-assisted debugging. The system is deployed using Kubernetes for scalability.
